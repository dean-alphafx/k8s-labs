## Accessing services from API 

Create an Nginx pod an expose the service: 

```
kubectl expose pod/nginx --port 80
```

The service can be accessed through kubectl proxy command:  

```
curl localhost:8001/api/v1/namespaces/default/services/nginx/proxy/
```
